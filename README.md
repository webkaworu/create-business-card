## サイト概要
名刺のテンプレートを選択し、テキストの色や配置、大きさを編集して、画像でダウンロードできるサイトです。

## 使用技術
### フロントエンド
- HTML
- CSS（SCSS）
- JavaScript
- Vue3
- Bootstrap
### バックエンド
- PHP 8.2
- Laravel 10.0
- composer
### 開発環境
- Docker/Docker-compose
